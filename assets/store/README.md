# Chrome Web Store Assets

This directory contains promotional images and screenshots for the Chrome Web Store listing.

## Directory Structure

- `screenshots/` - Extension screenshots (1280x800 or 640x400)
- `promotional/` - Promotional tiles and marquee images
- `icons/` - Store-specific icon variations if needed

## Required Assets

See [PROMOTIONAL_IMAGES.md](../../PROMOTIONAL_IMAGES.md) in the root directory for detailed requirements and specifications.

### Minimum Requirements for Store Submission:
1. Small promotional tile (440x280)
2. At least 1 screenshot (recommend 3-5)

## Asset Guidelines

- All images should be PNG or JPEG
- Follow Chrome Web Store image guidelines
- Use consistent branding and colors
- Ensure text is readable at small sizes
