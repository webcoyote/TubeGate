# Promotional Images & Screenshots Guide

This document outlines the image requirements for the Chrome Web Store listing.

## Required Images

### 1. Small Promotional Tile (Required)
- **Size:** 440x280 pixels
- **Format:** PNG or JPEG
- **Purpose:** Shown in Chrome Web Store search results and category pages
- **Content Suggestions:**
  - TubeGate logo with tagline
  - "Control Your YouTube Feed"
  - Clean, simple design with brand colors

### 2. Screenshots (Required - at least 1, up to 5 recommended)
- **Size:** 1280x800 or 640x400 pixels
- **Format:** PNG or JPEG
- **Aspect Ratio:** 16:10 preferred

#### Screenshot Ideas:

**Screenshot 1: Extension Popup - Filters Tab**
- Show the main interface with example filters
- Highlight the filter count
- Show the enabled toggle
- Caption: "Add unlimited keywords to filter unwanted content"

**Screenshot 2: Extension Popup - Settings Tab**
- Show the settings interface
- Highlight sync option
- Show Send Feedback link
- Caption: "Simple settings and optional cloud sync"

**Screenshot 3: Before/After Comparison**
- Split screen showing YouTube feed before and after filtering
- Before: cluttered feed with various content
- After: clean, curated feed
- Caption: "See filtered videos disappear in real-time"

**Screenshot 4: YouTube Homepage with Filter Active**
- Show YouTube homepage with some videos visible
- Subtle indicator that filtering is active
- Caption: "Works seamlessly on YouTube homepage and search"

**Screenshot 5: Filter Examples**
- Show the popup with organized filters using comments
- Example categories: sports, sponsored content, topics
- Caption: "Organize filters with comments for easy management"

### 3. Marquee Promotional Tile (Optional but Recommended)
- **Size:** 1400x560 pixels
- **Format:** PNG or JPEG
- **Purpose:** Featured placement on Chrome Web Store
- **Content Suggestions:**
  - Large TubeGate logo
  - Compelling tagline: "Take Control of Your YouTube Experience"
  - Key features: "Privacy-First • Easy to Use • Open Source"
  - Use brand colors (purple #667eea)

### 4. Additional Assets (Optional)

**Feature Graphic**
- **Size:** 1280x720 pixels
- Show key features with icons
- "Filter by Keywords • Privacy First • Sync Across Devices"

## Design Guidelines

### Color Scheme
- **Primary:** #667eea (purple)
- **Secondary:** #5568d3 (darker purple)
- **Accent:** #2c3e50 (dark blue-gray)
- **Background:** #f5f7fa (light gray)

### Typography
- Use clean, modern sans-serif fonts
- Ensure text is readable at small sizes
- High contrast for accessibility

### Content Guidelines
- Keep text minimal and impactful
- Focus on benefits, not just features
- Use real screenshots when possible
- Avoid cluttered designs
- Maintain consistent branding

## Screenshot Capture Instructions

### For Extension Popup Screenshots:

1. Open TubeGate extension popup
2. Set browser zoom to 100%
3. Capture screenshot using:
   - macOS: Cmd+Shift+4, then Space, click window
   - Windows: Use Snipping Tool or Snip & Sketch
   - Chrome DevTools: Device toolbar for consistent sizing

4. Crop to remove browser chrome if desired
5. Scale to required dimensions maintaining aspect ratio

### For YouTube Screenshots:

1. Navigate to YouTube homepage or search
2. Ensure TubeGate is active/inactive as needed
3. Use a clean browser profile (no other extensions visible)
4. Capture full page or relevant section
5. Annotate if needed to highlight filtering

## File Naming Convention

Use clear, descriptive names:
- `small-tile-440x280.png`
- `screenshot-1-popup-filters.png`
- `screenshot-2-popup-settings.png`
- `screenshot-3-youtube-filtering.png`
- `marquee-1400x560.png`

## Storage Location

Store all promotional images in the `assets/store/` directory:

```
assets/
  store/
    screenshots/
    promotional/
    icons/
```

## Image Optimization

- Use PNG for screenshots and images with text
- Use JPEG for photographic content
- Compress images to reduce file size
- Maintain quality (don't over-compress)
- Tools: TinyPNG, ImageOptim, Squoosh

## Validation Checklist

Before submitting to Chrome Web Store:

- [ ] Small tile is exactly 440x280 pixels
- [ ] At least 1 screenshot (recommend 3-5)
- [ ] Screenshots are 1280x800 or 640x400 pixels
- [ ] All images are PNG or JPEG
- [ ] File sizes are reasonable (<1MB each)
- [ ] Images clearly demonstrate extension functionality
- [ ] Text is readable and professional
- [ ] No spelling errors or typos
- [ ] Consistent branding across all images
- [ ] Images follow Chrome Web Store policies
